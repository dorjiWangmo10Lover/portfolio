National Digital Voting

Blockchain based voting system.

Project information

  • Category: Web design
  • Project date: 01 March, 2020

Project Overview

National Digital Voting (NDV) is a blockchain-based platform designed to facilitate secure, tamper-proof, and transparent online elections. It allows Bhutanese citizens to cast their votes digitally with confidence and privacy, enhancing democratic participation through technology.

My Role:

I was responsible for the frontend development of NDV. I designed the user interface using React and Bootstrap, implemented secure authentication, and collaborated on smart contract logic for immutable voting records on the blockchain.

Responsibilities:

• Designed a user-friendly multi-step voting flow.
• Developed components for voter login, vote casting, and results display.
• Integrated Ethereum smart contracts for decentralized vote recording.
• Ensured mobile responsiveness and accessibility across devices.

Pain Points:

1. Ensuring vote immutability while maintaining user privacy was a key challenge. We achieved this through cryptographic hashing and voter anonymization.

2. Balancing a simple UX for all users—including non-tech-savvy voters—while preserving blockchain security and compliance was complex.

Usability Study: Findings

• Users preferred one-click voting confirmation.
• Voter verification instructions needed simplification.
• Results display was improved to highlight winning candidates clearly.

Key Takeaways

  • Blockchain can enhance voting transparency and integrity.
  • User-friendly design is essential in civic tech solutions.
  • Security and simplicity must go hand-in-hand for public systems.
